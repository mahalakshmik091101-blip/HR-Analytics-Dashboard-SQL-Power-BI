HR Analytics Dashboard | SQL • Power BI 

🎯 The Challenge
HR leadership lacked a centralized, data-driven view of workforce health. Key questions around attrition, workforce cost, employee performance, and engagement were spread across static reports, limiting strategic decision-making.

________________________________________

🔎 The Approach

I analyzed 7,000 employee records using SQL to clean, transform, and calculate critical workforce KPIs.

I then built a Power BI dashboard using a star schema model and DAX measures to create dynamic, executive-level insights covering:

👥 Headcount & active workforce

🔁 Turnover rate

💰 Workforce cost & salary distribution

📊 Performance & satisfaction metrics

🌍 Demographic and job-level segmentation

________________________________________

📊 The Insights

The analysis revealed:

👥 19.89% turnover rate with 6K active employees out of 7K

💰 $202M annual workforce cost with $27.24K average salary

📈 Workforce growth stabilized post-2017, supporting long-term hiring forecasts

⭐ Strong correlation between manager evaluation scores and employee productivity

🏢 Entry & Associate roles represent the highest workforce concentration

💬 Engagement and satisfaction levels significantly influence performance outcomes
